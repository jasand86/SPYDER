Simple Structure

Implementing the grid is very straightforward and requires just three layers.

1) .grid – Serves as the parent container for the entire grid.

2) .grid__row – Helps set the vertical rhythym of layouts. This layer should always be placed within the .grid element.

3) .grid__col or .col--[x]-[y_z] – Use columns to build and control the horizontal flow of layouts. Columns should always be placed within the .grid__row element. ** It's important to note there are two syntax variations for columns—both will be explained in detail in the following section.

<div class="grid">
<div class="grid__row">
<div class="grid__col">
...Your site content goes here
</div>
</div>
</div>

Fluid Responsiveness

Spyder is based on the flexbox module and uses percent based widths. This allows for fluid resizing of columns and rows.
